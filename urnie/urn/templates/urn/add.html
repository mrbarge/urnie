URL Helpers
The following optional helpers can be inserted into the URL to modify it before redirection:
  • %v - Insert the value of the first parameter, as-is
  • %j - Translate parameters into a Jira JQL fuzzy text/comment search
  • %J - Translate parameters into a Jira JQL exact text/comment search
eg. an URN of "test" with a URL of "https://test.com/search?q=%v" will make "test hello" redirect to "https://test.com/search?q=hello"
